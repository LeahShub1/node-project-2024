## Installation

1. Clone the repository:
    ```sh
  git clone https://github.com/leahssss/node-project-2024.git
    ```
2. Navigate to the project directory:
    ```sh
    cd nodeProject
    ```
3. Install dependencies:
    ```sh
    npm install
    ```
